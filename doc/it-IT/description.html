Feedly Notifier è un'estensione che mantiene aggiornati i feed sottoscritti tramite il servizio di Feedly. Caratteristiche:
  • conteggio delle notizie non lette
  • titoli delle notizie non lette (con link diretto al sito della notizia) nella finestra popup
  • anteprima dell'articolo nella finestra popup
  • possibilità di scegliere aggiornamenti di singole categorie
  • possibilità di filtrare le notizie per categoria nella finestra popup
  • possibilità di segnare le notizie come già lette
  • notifiche sonore e per desktop
  • aggiunge la possibilità di sottoscrivere i feed usando l'apposita interfaccia di Firefox
