Feedly notifier on imepisike laiendus, mis hoiab sind kursis feedly tellimustega. Omadused:
  • lugemata uudiste arv
  • lugemata uudiste pealkirjad (koos otse lingina uudislehele) hüpikaknas
  • artikli eelvaade hüpikaknas
  • võimalus piirata uuendusi kategooriate abil
  • võimalus filtreerida uudiseid kategooria alusel hüpikaknas
  • võimalus märkida uudiseid loetuks
  • võimalus avada veebileht nupuvajutuse teel (töötab ainult kui loendur)
  • töölaua ja heli teadustused
  • lisab võimaluse uudisvoo tellimiseks läbi standardse Firefox dialoogi
Tugi: Kui oled leidnud vea või sul on mõni soovitus, palun anna meile teada. Raporteeri need palun siia: https://github.com/olsh/Feedly-Notifier-Firefox/issues või kontaktteeru e-posti teel. Aita meil tõlkida lisa (võta ühendust e-posti teel või saada tõmbe päring GitHub keskonnas).
