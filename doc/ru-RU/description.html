Feedly Notifier это расширение для чтения новостных подписок с Feedly.

Возможности:

  • счетчик непрочитанных сообщений
  • отображение заголовков непрочитанных сообщений (с ссылкой на статью) во всплывающем окне
  • просмотр статьи прямо во всплывающем окне
  • возможность устанавливать категории для обновления
  • возможность фильтровать новости по категории
  • возможность пометить статью как прочитанную
  • возможность работы только как счетчика новостей (при клике переходить на сайт feedly)
  • уведомления рабочего стола и звуковые оповещения о новых статьях
  • добавляет возможность подписываться на новости используя стандартный диалог
